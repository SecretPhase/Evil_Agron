* יש לוודא שהקובץ AGRON2006.csv יהיה באותו התיקייה עם הסקריפט.



חפשו את הסיסמא בטלגרם - האקרים בישראל


```shell
# חילוץ הקובץ עם הסיסמא

unzip AGRON2006.zip
```
